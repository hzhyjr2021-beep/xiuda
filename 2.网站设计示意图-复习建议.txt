页面名称：四科复习建议（标签式布局）

视觉布局
1. 顶部固定栏（高度 64px，白底+下阴影）：
   - 左侧返回按钮“← 首页”。
   - 中间标题“复习建议”。
   - 右侧显示“剩余 XX 天”小徽标。

2. 标签切换区（位于顶部栏下方）：
   - 横向五个同宽标签：政治 / 英语（一） / 数学（三） / 专业课 / 其他。
   - 选中标签底部橙色指示条，文字加粗。

3. 内容卡片（整体宽度 960px 居中）：
   - 背景白色，阴影柔和，圆角 16px；内部采用单列段落排版，直接呈现 Word 文档中的复习建议文本。
   - 各标签内容如下：

   政治
   1. 考前把选择题背诵小册子，再翻上几遍；
   2. 把肖四肖八选择题吃透，错题反复看知识点；
   3. 肖四主观题背熟；
   4. 可以下载公众号（小谭学长、利之学姐、、不易学长）的时政资料；

   英语（一）
   1. 背诵作文模版，大小作文必须保持练习，至少2/3天就要写一篇；
   2. 保持手感，优先级是：写作>阅读>翻译>新题型>完形填空；
   3. 单词背真题核心词，每天都要背，一直背到考前；
   4. 模拟考试，把控时间，安排好每个题型的做题顺序；

   数学（三）
   1. 掐时间去模考，适应真题的风格；
   2. 模拟卷可以补充李林6+4；
   3. 回顾错题，错题反复做；
   4. 反复回顾知识点，一定要查漏补缺；

   专业课
   1. 熟练背诵简答题；
   2. 核心考点一定要重点反复记忆；
   3. 搞懂真题，抓住复习的重点，专业课真题里面的错题，必须要掌握；
   4. 注重知识点的回顾，把教材从头到尾快速的翻一翻；

   其他
   1. 最后阶段，一定要从自己的实际需求出发复习；
   2. 定期及时回顾知识点；
   3. 考前一个月左右一定要根据考试时间安排起床与学习时间；
   4. 黎明之前最黑暗，放弃就是大笨蛋！

4. 底部操作区：
   - “切换科目”提示箭头或标签左右切换按钮。

5. 背景与间距：
   - 整体背景浅灰，留足呼吸空间；不同卡片之间保持 32px 间隔，确保文本易读。
